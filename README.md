# Nodeplot

Nodeplot is a visual tool for analyzing data and creating graphs using a node-based interface. It allows you to import CSV data, process it through various nodes, and visualize the results.

## Key Features

*   **Visual Node Editor**: Create data flows by connecting nodes.
*   **CSV Import**: Easily load and view CSV files.
*   **Interactive Controls**: Create custom interfaces to control your data parameters.
*   **Data Visualization**: Plot your data using powerful charting tools.

## Prerequisites

Before you begin, ensure you have the following installed on your computer:

*   [Node.js](https://nodejs.org/) (Version 18 or higher recommended)

## Installation

1.  **Open a terminal** (Command Prompt, PowerShell, or Terminal).
2.  **Navigate** to the project folder.
3.  Run the following command to install the necessary dependencies:

    ```bash
    npm install
    ```

## Running the Application

To start the application locally:

1.  Run the development command:

    ```bash
    npm run dev
    ```

2.  The terminal will show a local URL (usually `http://localhost:5173/`).
3.  **Click the link** or copy and paste it into your web browser to open Nodeplot.
